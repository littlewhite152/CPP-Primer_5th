- # 容器set

![容器set](https://github.com/havenow/my-C-plus-plus/blob/master/STL%E6%A0%87%E5%87%86%E5%BA%93%E4%B8%8E%E6%B3%9B%E5%9E%8B%E7%BC%96%E7%A8%8B/images/%E5%AE%B9%E5%99%A8set.png)  

从set里面拿一个迭代器的时候，拿到的是一个const的iterator，因此set不能使用迭代器改它的内容。  

set的其实也是一个adapter，所有的操作都是靠低层的红黑树完成。
