insert返回第一个被插入元素的迭代器

copy返回的是其目的位置迭代器（递增后）的值

erase返回被删最后一个元素之后的迭代器



|                    | vector                   | string                   | list | forward_list | deque | array | 备注                             |
| ------------------ | ------------------------ | ------------------------ | ---- | ------------ | ----- | ----- | -------------------------------- |
| insert之后的迭代器 | 失效                     | 失效                     | 未   | 未           | 失效  | 无    |                                  |
| erase之后的迭代器  | 被删位置之后的迭代器失效 | 被删位置之后的迭代器失效 | 未   | 未           | 未    | 无    | 删除首尾之外的元素会导致都会失效 |
